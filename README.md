Waifu2xBarracuda
================

**Waifu2xBarracuda** is a [Waifu2x] implementation with the Unity [Barracuda]
NN inference library.

[Waifu2x]: https://github.com/nagadomi/waifu2x
[Barracuda]: https://docs.unity3d.com/Packages/com.unity.barracuda@latest

Acknowledgement
---------------

### ONNX files
